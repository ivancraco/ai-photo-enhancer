1. Información del Responsable

AI Photo Enhancer es operada desde la República Argentina. Esta Política describe cómo recopilamos, utilizamos y protegemos la información cuando los usuarios utilizan la aplicación.

2. Datos que Recopilamos

2.1 Imágenes proporcionadas por el usuario

  • Las imágenes se utilizan exclusivamente para aplicar mejoras mediante inteligencia artificial.
  • Se almacenan temporalmente con fines de procesamiento.
  • No se utilizan para entrenar modelos de IA.

2.2 Identificadores técnicos anónimos

  • Se generan identificadores anónimos para permitir el funcionamiento del servicio.
  • No permiten identificar personalmente al usuario.

2.3 Datos técnicos de uso

  • Tipo de dispositivo, sistema operativo e idioma.
  • Datos de rendimiento y errores.

2.4 Compras y suscripciones

  • Procesadas por la tienda de aplicaciones.
  • No almacenamos información financiera.

2.5 Publicidad

  • Se muestran anuncios mediante redes publicitarias externas.
  • Estas pueden usar identificadores del dispositivo para personalización.

3. Finalidad del Uso de Datos

  • Procesar imágenes solicitadas por el usuario.
  • Proveer funcionalidades premium.
  • Mostrar anuncios en la versión gratuita.
  • Mejorar estabilidad y rendimiento.
  • Detectar fraude o uso indebido.

4. Retención y Eliminación de Datos

  • Las imágenes se eliminan automáticamente dentro de un plazo máximo de 24 horas.
  • No se almacenan de forma permanente.
  • Los proveedores externos aplican la misma política de eliminación.

5. Procesamiento por Terceros

Utilizamos proveedores tecnológicos que procesan datos únicamente para operar el servicio bajo obligaciones de confidencialidad.

6. Transferencias Internacionales

Los datos pueden procesarse en servidores fuera del país del usuario aplicando salvaguardas de seguridad adecuadas.

7. Seguridad

  • Cifrado en tránsito mediante HTTPS.
  • Acceso restringido a los sistemas.
  • Eliminación automática de datos temporales.
